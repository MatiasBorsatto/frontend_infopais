# INFOPAIS

## Tecnologías utilizadas:

**Frontend:**

- Vue.js 3
- Pinia (State Management)

**Backend:**

- Node.js
- Express.js

**Librerías UI:**

- PrimeVue
- Tailwind CSS

**Seguridad:**

- bcrypt (Hash de contraseñas)
- JWT (JSON Web Tokens)

**Base de datos:**

- PostgreSQL
- Sequelize (ORM)
